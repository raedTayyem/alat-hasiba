### License Validation (No Database)

#### Client-Side Domain Whitelisting
```javascript
// Simple license check (no server required)
const LICENSE_CONFIG = {
  key: "CUSTOMER_LICENSE_KEY_HERE",
  domains: ["client-domain.com", "www.client-domain.com"],
  expiryDate: "2027-01-01",
  tier: "single-domain"
};

function validateLicense() {
  const currentDomain = window.location.hostname;
  const today = new Date();
  const expiry = new Date(LICENSE_CONFIG.expiryDate);

  if (!LICENSE_CONFIG.domains.includes(currentDomain)) {
    console.error("Invalid domain for license");
    return false;
  }

  if (today > expiry) {
    console.error("License expired");
    return false;
  }

  return true;
}
```

**Pros:**
- No backend infrastructure needed
- Fast validation (no API calls)
- Works offline
- Simple to implement

**Cons:**
- Can be bypassed by technical users
- No real-time license revocation
- Requires code changes per customer

---

#### API License Validation (via Gumroad/External Service)
```javascript
// Validate via API (requires backend or Gumroad)
async function validateLicense(licenseKey, domain) {
  const response = await fetch('https://api.gumroad.com/v2/licenses/verify', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      product_id: 'YOUR_PRODUCT_ID',
      license_key: licenseKey,
      increment_uses_count: true
    })
  });

  const data = await response.json();

  if (data.success && data.purchase.custom_fields?.domain === domain) {
    return true;
  }

  return false;
}
```

**Pros:**
- Real-time validation
- Can revoke licenses remotely
- Tracks usage
- Professional licensing

**Cons:**
- Requires internet connection
- Gumroad dependency
- Slightly more complex

**Recommended:** Use Gumroad License API for paid tiers
